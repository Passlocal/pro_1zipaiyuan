# 易拍选 (EpicShot) V1.2.0 UI/UX 设计文档

## 设计原则

**简洁方便用户体验为第一重点**。在移动端，用户打开页面即见核心功能，不被无关信息遮挡。

## 响应式规范

### 断点

| 断点 | 范围 | 目标 |
|------|------|------|
| 移动端 | ≤768px | 手机竖屏 |
| 平板 | 769-1024px | 平板横屏 |
| 桌面端 | ≥1025px | 桌面显示器 |

### 移动端规则

1. **触控热区**: 所有可点击元素 ≥44px × 44px
2. **输入框**: 字体 ≥16px（防止iOS自动缩放）
3. **弹窗**: 宽度统一为 `94vw`
4. **按钮**: 全宽或最小44px高度
5. **表格**: 超出部分横向滚动，不换行
6. **卡片网格**: 桌面3列→平板2列→移动端1列

### 页面布局规范

| 页面 | 桌面布局 | 移动端布局 |
|------|----------|------------|
| 登录 | 左右分屏(40/60) | 品牌隐藏，表单全屏 |
| 注册 | 居中卡片 | 全宽卡片，无阴影 |
| 看板 | 顶部筛选+卡片网格 | 筛选横向滚动，单列 |
| 项目详情 | 左右分栏 | 工具栏固定底部 |
| 客户视图 | 浮动工具栏 | 简化工具栏 |
| 作品集编辑 | 左右分栏(预览/工具) | 上下排列 |
| 工作空间 | 表单+表格 | 表单纵向，表格横向滚动 |

## 移动端登录页重设计（V1.2.0）

这是本轮最重要的UI优化：

**问题**: 移动端品牌面板(300-400px)占据首屏，登录表单在下方不可见。

**方案**:
- 桌面端: 保持左右分屏
- 移动端(≤767px): 隐藏品牌面板，仅显示紧凑品牌标识(易拍选+标语)，登录表单直接可见

**效果**: 用户打开页面第一眼看到的就是登录表单。
