# 易拍选 (EpicShot) V1.2.0 运营手册

## 产品定位

**易拍选** — 专业摄影审片协作平台。

目标用户：摄影工作室（摄影师 + 后期团队 + 客户）

核心价值：告别微信轰炸，用专业审片流程替代混乱的微信群聊。

## 用户角色

| 角色 | 功能 | 典型场景 |
|------|------|----------|
| 工作室owner | 创建项目、管理成员、查看报表 | 摄影工作室老板 |
| 工作室editor | 上传照片、处理批注、确稿 | 摄影师/后期修图师 |
| 客户 | 查看照片、批注意见、确稿 | 被拍摄的客户 |

## 核心流程

### 1. 摄影工作室入驻
```
注册 → 创建工作空间 → 邀请成员 → 创建项目 → 上传照片
```

### 2. 客户审片
```
工作室分享链接 → 客户打开(无需下载APP) → 浏览照片 → 批注意见 → 确稿
```

### 3. AI 辅助
```
上传照片 → AI色差检查 → AI一致性检查 → AI风格样片生成 → 确稿交付
```

## 功能模块

### 项目管理
- 创建/编辑/删除项目
- 拍摄单元（分组管理照片）
- 批量上传照片
- 项目状态（活动/完成/归档）
- 预警设置

### 审片协作
- 图片查看器（缩放/旋转/对比）
- 批注工具（矩形/圆形/箭头/文字）
- 评论卡片（待处理/处理中/已解决/争议）
- 版本对比（修改前后）

### 客户视图
- 分享链接访问（无需登录）
- 移动端优化的浏览体验
- 批注与确稿
- 修改请求

### AI 功能
- 色差巡检（色彩一致性检查）
- 一致性检查（多张照片风格统一）
- 风格样片生成（AI预演修图效果）
- 费用：24h缓存 + 50次/天/项目 + 月≥200次预警

### 数据看板
- 项目健康度总览
- 成员效率分析
- 财务分析报表
- 数据导出

### 作品集
- 精选照片编辑
- 封面/标题/描述
- 分享二维码
- 客户浏览统计

## 通知系统

支持三种通知渠道：

| 渠道 | 状态 | 说明 |
|------|------|------|
| 站内通知 | 已实现 | 实时通知列表 |
| 邮件通知 | 需配置SMTP | 任务分配、争议提醒 |
| 微信通知 | 预留 | 需配置微信公众号 |

## 外部服务对接状态

| 服务 | 状态 | 影响 |
|------|------|------|
| AI 服务 | 需配置 | 无API Key时使用内置fallback |
| 微信OAuth | 预留 | 当前未接入登录UI，不影响使用 |
| SMTP邮件 | 需配置 | 邮件通知不可用 |
| SSL证书 | 需配置 | 生产环境必须 |

## 运营建议

### 首月上线
1. 先配置 SMTP 邮件服务（通知功能）
2. 配置 SSL 证书（HTTPS 安全）
3. AI 服务按需配置（成本可控后再启用）

### 客户推广
1. 项目完成后分享链接给客户
2. 客户无需下载APP，手机浏览器即可审片
3. 作品集可生成二维码，方便传播

### 成本控制
- AI 功能内置多层成本控制，无需担心滥用
- 24小时缓存避免重复生成
- 每月≥200次自动预警
