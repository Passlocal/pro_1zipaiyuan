# 易拍选（EpicShot）V1.2.0 内测指引

---

## 一、内测目标

| 维度 | 目标 |
|------|------|
| 参与工作室 | 3-5 家，不同规模 |
| 完整项目数 | ≥ 10 个（创建→确稿交付） |
| 周期 | 2-3 周 |
| 日活跃率 | ≥ 80% |

---

## 二、工作室招募筛选标准

1. **规模分布**：5 人以下 1-2 家，5-15 人 1-2 家，15 人以上 1 家
2. **业务类型**：至少覆盖婚纱、商业摄影、电商产品 3 个品类
3. **当前痛点**：正在使用微信+网盘协作流程，对效率有明确诉求
4. **配合意愿**：愿意每周接受 30 分钟电话回访

---

## 三、电话回访模板（每周一次）

### 开场（2 分钟）
> 感谢您参与易拍选内测。这周使用下来，整体感觉怎么样？

### 核心问题（20 分钟）

**老板视角：**
1. 这周战情室打开了几次？主要看什么信息？
2. 有没有哪些信息你想看但战情室里没有的？
3. 这周有没有项目出现延迟/争议？你是怎么发现的？

**修图师视角：**
1. 待办列表每天都看吗？优先级排序准不准？
2. 快速流转（标记已解决）操作你用过吗？感觉怎么样？
3. 标注工具在手机上用着顺手吗？有没有卡顿？

**客户视角：**
1. 第一次打开分享链接时，知道该做什么吗？需要额外引导吗？
2. 确稿流程走下来顺不顺畅？
3. 有没有遇到过不知道该点哪里的情况？

### 收尾（8 分钟）
> 如果让你给这周的使用体验打分（1-10 分），你打几分？扣分在哪里？
> 你觉得易拍选能不能替代你现在的微信+网盘流程？差别在哪里？

## 四、NPS 调研模板（内测结束时）

```
1. 你有多大可能向同行推荐易拍选？（0-10 分）
2. 你给这个分数的原因是什么？
3. 易拍选能不能替代你现在的协作工具？
   □ 完全可以 □ 基本可以 □ 勉强可以 □ 不太行
4. 你觉得最需要改进的 3 个地方是？
5. 如果易拍选收费，你愿意每月支付多少？（可选）
```

## 六、异常监控指标

| 指标 | 告警阈值 | 监控方式 |
|------|----------|----------|
| AI 生成失败率 | > 5% | Sentry / 日志 |
| 图片加载超时 | > 1 次/用户/天 | 前端埋点 |
| API 5xx 错误 | > 0.1% | Sentry |
| 页面崩溃 | 任何一次 | Sentry |
| 移动端操作卡顿反馈 | 用户主动反馈 | 电话回访 |
