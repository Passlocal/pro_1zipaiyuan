## 1. 产品概述

**易拍选（EpicShot）** 是一款面向商业摄影行业的在线协作平台，帮助摄影师、修图师与客户之间高效完成图片审阅、标注修改、色差巡检和成片交付。

### 核心功能

| 功能模块 | 说明 |
|---------|------|
| 战情室 | 老板每日工作起始页，项目健康度一目了然，成员负载实时监控 |
| 项目管理 | 创建和管理摄影项目，跟踪项目进度，支持流程模板一键创建 |
| 图片标注 | 使用画笔、箭头、矩形、椭圆、文字等工具在图片上精确标注修改意见 |
| 任务指派 | 将意见卡片指派给特定修图师，争议预警自动标记反复重开的问题 |
| 色差巡检 | 一键检测全组图片的色温、亮度和对比度偏差，支持场景化修正 |
| 光影一致性巡检 | 跨场景分析光影逻辑一致性，检测灯位偏移、高光不一致等问题 |
| 智能批量筛选 | 输入行话关键词，AI 自动匹配相关图片，一键应用筛选建议 |
| 作品集 | AI 辅助生成精美作品集，支持手机框架预览和公开展示 |
| 客户端分享 | 生成免登录分享链接，客户可直接查看项目并确认确稿 |
| 保护型确稿 | 确稿后锁定编辑，客户需申请修改由老板审批驳回 |
| 通知系统 | 分级预警通知，临期、指派、争议、确稿全场景覆盖 |
| 修改溯源 | 完整记录每次修改和标注的时间轴，支持版本前后对比 |
| 团队协作 | 工作空间成员管理，支持多人实时在线协作 |

### 技术亮点

- 支持 JPG / PNG / WebP / SVG 等多种图片格式
- 单文件最大支持 50MB
- 自动生成多级缩略图（320px / 800px / 原图），优化加载速度
- 支持 HTTPS 加密传输
- 基于 JWT 的身份认证，保障数据安全
- WebSocket 实时协作（标注同步、光标位置共享）
- 结构化 JSON 日志，四级分层，支持请求追踪

## 2. 快速入门

### 2.1 注册账号

1. 在浏览器中打开系统地址，进入登录页面
2. 点击页面底部的「还没有账号？立即注册」链接，跳转到注册页面
3. 填写以下信息：
   - **姓名**：您的真实姓名或花名（必填）
   - **邮箱**：用于登录和接收通知的有效邮箱地址（必填）
   - **密码**：至少 6 个字符（必填）
   - **确认密码**：再次输入密码以确认（必填）
4. 系统会自动校验：
   - 邮箱格式是否正确
   - 两次输入的密码是否一致
   - 密码长度是否满足最低要求（≥6 位）
5. 点击「注册」按钮提交
6. 注册成功后，系统会自动跳转到登录页面

### 2.2 登录系统

#### 邮箱密码登录

1. 在登录页面，输入您的**邮箱地址**和**密码**
2. 点击密码输入框右侧的眼睛图标，可以切换密码的显示/隐藏
3. 点击「登录」按钮
4. 登录成功后，系统会跳转到战情室页面

#### 忘记密码

如果忘记密码，请点击登录页面的「忘记密码？」链接，系统会提示您联系管理员重置密码。

### 2.3 界面概览

登录成功后，您将看到系统主界面，由以下区域组成：

- **左侧导航栏**：包含「战情室」「我的待办」「项目看板」「工作空间」等入口，底部显示当前套餐类型
- **顶部栏**：显示面包屑导航、通知铃铛（未读计数）、用户头像和退出按钮
- **内容区域**：根据当前页面显示对应的功能模块

左侧导航栏支持**折叠/展开**，点击侧边栏顶部的 ◀/▶ 按钮即可切换，折叠后可获得更大的内容显示空间。

## 3. 战情室（老板工作台）

战情室是老板和项目总监的每日工作起始页，登录后默认进入。集中展示所有项目的健康状态、成员负载和待处理事项。

### 3.1 统计面板

页面顶部展示五个核心统计卡片：

| 统计项 | 说明 |
|--------|------|
| 全部项目 | 当前空间内的项目总数 |
| 进行中 | 状态为「修改中」和「待确稿」的项目数 |
| 红灯项目 | 存在争议或严重超期的项目数 |
| 黄灯项目 | 临期预警或存在轻度问题的项目数 |
| 待处理 | 需要老板关注和决策的事项数 |

### 3.2 项目健康度

项目列表以卡片形式展示每个项目的健康状态：

- **绿灯** 🟢：项目进展正常，无预警
- **黄灯** 🟡：临近截止日期或存在轻微问题，需要关注
- **红灯** 🔴：存在争议卡片、严重超期，需要立即处理

每个项目卡片显示：
- 项目名称和客户名称
- 当前状态标签
- 待处理问题数
- 预警原因标签（如"临期""争议"）
- 截止日期

### 3.3 成员负载

页面底部展示团队成员的工作负载：

- 每个成员的**姓名和头像**
- **当前任务数**（被指派的卡片数量）
- **负载进度条**（绿色=正常，黄色=繁忙，红色=超负荷）

### 3.4 一键催稿

在战情室红灯/黄灯项目卡片上，提供「提醒客户」按钮。

- 点击后系统自动向客户发送预置提醒通知
- 卡片上显示「已催稿」标签和最近催稿时间
- 客户收到通知后，催稿计数更新

### 3.5 健康度规则说明

战情室页面顶部有「健康度规则」问号图标。

- 点击弹出浮窗，列出红灯、黄灯的触发条件
- 红灯：严重超期、客户卡顿、争议存在
- 黄灯：轻度超期、回复延迟
- 浮窗内容与后台逻辑一致

### 3.6 成员负载数值显示

在成员负载进度条旁显示「当前任务数 / 建议上限」。

- 格式示例：`5 / 15`
- 建议上限由老板在空间设置中自定义，默认 15
- 数字随任务数实时变化

## 4. 修图师待办列表

点击左侧导航栏的「我的待办」进入，是修图师的个人工作台。

### 4.1 任务优先级

待办列表按优先级自动排序：

| 优先级 | 标识 | 说明 |
|--------|------|------|
| 争议 | 🔴 红色标记 | 被反复重开的卡片，需要重点关注 |
| 临期 | 🟡 黄色标记 | 所属项目临近截止日期 |
| 普通 | ⚪ 默认 | 常规待处理卡片 |

每条待办显示：
- 关联图片缩略图
- 项目名称和产品单元
- 标注内容摘要
- 指派时间
- 优先级标签

### 4.2 快速标记已解决

1. 在待办卡片上点击「标记为已解决」
2. 卡片从待办列表移除
3. 关联项目中的意见卡片状态同步更新
4. 指派者收到完成通知

### 4.3 快速流转模式

标记「已解决」后，系统自动将下一条待办卡片高亮并滚动至视图中央。

- 修图师可连续操作，无需手动切换
- 提升批量处理效率

### 4.4 任务预览浮窗

鼠标悬停或点击待办卡片缩略图，弹出放大预览浮窗。

- 浮窗尺寸 280×210px，居中展示标注区域
- 标注区域清晰可见，加载流畅

### 4.5 预计耗时字段

待办卡片显示预计耗时标签。

- 选项：15分钟 / 30分钟 / 1小时
- 在意见卡片指派时选填
- 修图师可按预计耗时排序

## 5. 工作空间管理

工作空间是您团队协作的基本单位。点击左侧导航栏的「工作空间」进入管理页面。

### 5.1 空间基本信息

- **空间名称**：点击名称区域可直接编辑，修改后即时生效
- **Logo 上传**：点击 Logo 区域可上传自定义品牌标识，支持 JPG / PNG / SVG 格式
- **套餐类型**：显示当前套餐（免费版 / 专业版 / 企业版）
- **创建时间**：显示工作空间的创建日期

### 5.2 成员管理

在工作空间页面的「成员管理」区域，您可以：

- **查看成员列表**：显示所有成员的姓名、邮箱、角色（拥有者/编辑者）和状态（活跃/已禁用）
- **邀请成员**：点击「邀请成员」按钮，输入被邀请人的邮箱地址，系统会发送邀请通知
- **修改角色**：点击成员行中的角色标签，可在「拥有者」和「编辑者」之间切换
  - **拥有者**：拥有空间的所有管理权限，包括删除空间、管理成员等
  - **编辑者**：可以进行项目创建、编辑、标注等操作，但不能管理空间设置
- **移除成员**：点击成员行中的「移除」按钮，可将该成员移出工作空间
- **禁用成员**：可将成员状态设为「已禁用」，禁用后该成员无法登录和访问空间

### 5.3 API 密钥管理

> 此功能面向开发人员，用于通过 API 对接外部系统。

在工作空间页面可查看和复制当前空间的 API 密钥，用于第三方系统集成。

## 6. 项目看板

项目看板是日常项目管理的核心页面。点击左侧导航栏的「项目看板」进入。

### 6.1 统计概览

看板顶部显示四个统计卡片：

| 统计项 | 说明 |
|--------|------|
| 全部项目 | 当前空间内的项目总数 |
| 进行中 | 状态为「修改中」和「待确稿」的项目数 |
| 待处理 | 状态为「待确认」和「草稿」的项目数 |
| 已完成 | 状态为「已完成」和「已归档」的项目数 |

### 6.2 创建项目

1. 点击看板右上角的「新建项目」按钮
2. 在弹出的对话框中填写项目信息：
   - **项目名称**：为项目起一个易于识别的名称（必填）
   - **客户名称**：客户的姓名或公司名称（选填）

## 9. 光影一致性巡检

> V1.1 新增功能（BETA）

光影一致性巡检（光影探照灯）是高级品控工具，超越单张图片的色差分析，专注于同场景组图的光影逻辑一致性。

### 9.1 一键巡检

1. 在项目详情页顶部工具栏，点击「光影巡检」按钮
2. 进入光影一致性巡检页面
3. 点击「一键巡检」开始分析
4. 进度条实时显示分析进度
5. 完成后展示完整巡检报告

### 9.2 场景组分析

系统按产品单元将图片分组，分析每组的光影特征：

- **主光源方向**：检测每组图片的主光源来向（左上方/右上方/正上方/左前方/右前方）
- **一致性判定**：标记每组为「一致」或「不一致」
- **一致性评分**：0-100 分综合评分，卡片颜色渐变（绿→黄→红）

### 9.3 异常报告

系统检测以下 6 类光影异常：

| 异常类型 | 严重度 | 说明 |
|---------|--------|------|
| 光源方向 | 高 | 主光源方向不一致，疑似灯位偏移 |
| 高光位置 | 高 | 面部高光位置与组图其他图片不一致 |
| 阴影角度 | 中 | 阴影角度偏差超过阈值，光源高度可能变化 |
| 色温 | 中 | 同组图片色温不统一，白平衡设置可能变更 |
| 曝光 | 低 | 曝光量差异明显，闪光灯功率可能不一致 |
| 反光 | 低 | 反光面亮度差异过大，柔光设备位置可能偏移 |

每条异常包含：
- 严重度标签（高/中/低，颜色区分）
- 异常描述
- 异常图片与正常参考图片的并排对比
- 具体的修复建议

### 9.4 导出报告

点击「导出报告」按钮，支持打印或导出 PDF 格式的巡检报告。

### 9.5 BETA 安全声明

巡检入口和报告顶部增加文案声明。

- 内容：「BETA 功能：巡检结果仅供参考，不会自动修改任何图片。」

## 11. 客户端分享与确稿

客户端分享功能允许客户在不注册账号的情况下查看项目进度并提交确稿。

### 11.1 生成分享链接

1. 在项目详情页顶部工具栏，点击「分享」按钮
2. 设置分享链接的有效期：
   - **永久有效**：链接永久有效
   - **7 天有效**：链接 7 天后自动失效
   - **30 天有效**：链接 30 天后自动失效
3. 点击「生成链接」复制分享 URL
4. 将链接发送给客户

### 11.2 客户端查看项目

客户通过分享链接进入后，看到的页面包含：

- **项目名称和状态**：顶部显示项目名称和当前状态
- **进度条**：直观展示项目整体完成进度
- **图片查看器**：支持前后翻页浏览所有图片
- **倒计时/预计完成时间**：显示项目预计完成时间
- **意见卡片面板**：查看所有标注和修改意见
- **确稿确认**：确认对当前版本满意后，点击确稿按钮

### 11.3 保护型确稿流程

> V1.1 新增功能

保护型确稿防止客户在确稿后随意修改需求，确保团队工作成果不被随意推翻。

**确稿步骤**：
1. 客户浏览所有图片和修改意见
2. 确认所有修改已按要求完成
3. 点击「确稿」按钮
4. 系统弹出对比滑块确稿弹窗，展示修改前后对比
5. 客户再次确认后，项目状态变更为「已完成」
6. **确稿后项目进入只读状态**，所有编辑操作被锁定

**申请修改**：
1. 确稿后如需修改，客户点击「申请修改」
2. 填写修改原因
3. 系统发送通知给项目负责人（老板）
4. 只有老板角色可以驳回确稿，重新开放编辑

**驳回确稿**：
1. 老板在项目详情页看到修改申请
2. 审核修改原因
3. 点击「驳回确稿」，项目状态恢复为「修改中」
4. 团队成员可以继续编辑
5. 客户收到驳回通知

### 11.4 首次打开气泡引导

新客户首次打开分享链接时，出现 3 个轻量气泡引导：

- 「双指缩放查看细节」
- 「点击画笔开始标注」
- 「写完意见点发送」
- 点击可跳过，关闭后不再显示

### 11.5 二次确稿弹窗

客户点击确稿后，先出对比弹窗，确认后再弹出二次提醒。

- 二次提醒：「确稿后如需修改需联系摄影师申请。确认提交吗？」
- 两次确认，防止误操作

### 11.6 确稿对比高亮修改区域

确稿对比弹窗中，修改过的区域用半透明虚线框高亮。

- 附简要文字说明（如「曝光+0.5」）
- 高亮准确，方便客户核实修改效果

### 11.7 申请修改必须填写理由

客户申请修改时，必须填写不少于 5 个字的原因。

- 不满 5 字时无法提交
- 系统提示「修改原因至少5个字」

### 11.8 意见已读回执

客户浏览意见卡片及关联标注区域超 3 秒，自动标记「对方已读」。

- 工作室端显示「✓ 对方已读」及时间
- 支持筛选未读意见

## 13. 修改溯源时间轴

时间轴功能完整记录了项目从创建到交付的所有修改历史，便于追溯和复盘。

### 13.1 时间轴节点

时间轴按时间顺序展示以下类型的节点：

| 节点类型 | 触发事件 | 展示内容 |
|---------|---------|---------|
| 标注 | 创建标注 | 用户头像、标注工具类型、标注时间 |
| 指派 | 指派修图师 | 指派人、被指派人、指派时间 |
| 争议 | 卡片被标记为争议 | 争议原因、涉及卡片 |
| 修订 | 上传修改后的图片 | 修改前后对比缩略图、参数变更摘要 |
| 确认 | 确稿/解决卡片 | 确认人、确认时间 |
| 驳回 | 驳回确稿 | 驳回人、驳回原因、驳回时间 |
| 状态变更 | 项目状态变化 | 变更前状态 → 变更后状态、变更时间 |

每个节点包含：
- 用户头像
- 用户姓名
- 操作描述
- 时间戳

### 13.2 版本对比

点击时间轴中的修订节点，可打开版本对比弹窗：

- **修改前/后对比**：左右并排显示修改前后的图片
- **叠加滑块**：拖拽滑块查看修改前后的变化差异
- **参数变更详情**：列出具体的修改参数和数值变化

### 13.3 导出时间轴

- **日期筛选**：可按日期范围筛选时间轴节点
- **导出 PDF**：将完整时间轴导出为 PDF 文档，方便归档

### 13.4 叠加对比模式

版本对比弹窗支持三种模式切换：

- **左右并排**：修改前后图片并排显示
- **滑块对比**：拖动滑块查看修改前后差异
- **叠加对比**：上层透明度可调，精确对比差异

### 13.5 修改区域高亮开关

对比弹窗中增加开关。

- 开启后修改区域用虚线框标记
- 方便快速定位所有修改位置

## 18. 常见问题

### Q1：忘记密码怎么办？

请联系您的工作空间管理员重置密码。系统暂不支持自助重置密码功能。

### Q2：支持哪些图片格式？

支持 JPG、JPEG、PNG、WebP、SVG、GIF 格式。单文件最大 50MB。

### Q3：如何邀请团队成员？

进入「工作空间」页面，在成员管理区域点击「邀请成员」，输入对方的邮箱地址即可。

### Q4：项目状态如何变更？

项目状态会随着操作自动流转：
- 上传图片后可从「草稿」手动变更为「待确认」
- 客户提交意见后自动变为「修改中」
- 修改完成后可手动变更为「待确稿」
- 客户确稿后自动变为「已完成」
- 可在项目设置中手动归档

### Q5：确稿后还能修改吗？

确稿后项目进入只读状态。如需修改：
1. 客户点击「申请修改」提交修改原因
2. 老板审核后点击「驳回确稿」
3. 项目恢复为「修改中」状态，团队可继续编辑

### Q6：分享链接失效了怎么办？

在项目详情页重新生成分享链接即可。建议根据项目周期设置合适的有效期。

### Q7：标注会丢失吗？

标注数据使用相对坐标系统存储，与图片缩放比例无关。所有标注自动保存到服务器，不会因浏览器刷新或关闭而丢失。

### Q8：如何查看谁做了哪些修改？

进入项目详情页，点击「时间轴」按钮，可查看完整的修改溯源记录，包括标注、指派、修订、争议、确稿和状态变更的完整历史。

### Q9：什么是争议预警？

当一张意见卡片被反复重开 ≥2 次时，系统自动标记为「争议」。争议卡片红色高亮显示，并推送通知给项目负责人。战情室中对应项目显示红灯。

### Q10：作品集可以设置密码访问吗？

当前版本暂不支持密码访问，作品集发布后为公开访问。如需限制访问，建议使用分享链接的有效期功能。

### Q11：浏览器兼容性如何？

推荐使用最新版 Chrome、Edge、Firefox 或 Safari 浏览器。不支持 IE 浏览器。

### Q12：数据安全如何保障？

- 所有数据传输使用 HTTPS 加密
- 用户密码使用 bcrypt 12 轮哈希存储
- 身份认证使用 JWT Token，有效期 7 天
- 生产环境强制要求配置 JWT_SECRET 密钥
- 速率限制防止暴力破解
- 所有数据库查询使用参数化查询，防止 SQL 注入
- 输入严格校验，防止 XSS 攻击

|------|------|

| 速率限制 | 登录 30次/15min，全局 1000次/15min |
| 登录过期预警 | JWT 过期前 5 分钟提醒续期 |
| 外部协作者角色 | 细粒度权限隔离，仅访问被指派的项目 |

| 通知偏好设置 | 用户可独立控制站内/邮件/微信通知 |
| 通知快捷处理 | 铃铛下拉列表中直接审批操作 |
