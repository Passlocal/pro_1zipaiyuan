## 1. 产品概述

**易拍选（EpicShot）** 是一款面向商业摄影行业的在线协作平台，帮助摄影师、修图师与客户之间高效完成图片审阅、标注修改、色差巡检和成片交付。

### 核心功能一览

| 功能模块 | 说明 |
|---------|------|
| 战情室 | 老板每日工作起始页，项目健康度一目了然，成员负载实时监控 |
| 项目管理 | 创建和管理摄影项目，支持流程模板一键创建，多维度排序筛选 |
| 图片标注 | 画笔、箭头、矩形、椭圆、文字、橡皮擦六种工具，精确标注修改意见 |
| 任务指派 | 将意见卡片指派给特定修图师，争议预警自动标记反复重开的问题 |
| 色差巡检 | 一键检测全组图片的色温、亮度和对比度偏差，支持场景化修正 |
| 光影一致性巡检 | 跨场景分析光影逻辑一致性，检测灯位偏移、高光不一致等问题 |
| 智能批量筛选 | 输入行话关键词，AI 自动匹配相关图片，一键应用筛选建议 |
| 作品集 | AI 辅助生成精美作品集，支持手机框架预览和公开展示 |
| 客户端分享 | 生成免登录分享链接，客户可直接查看项目并确认确稿 |
| 保护型确稿 | 确稿后锁定编辑，客户需申请修改由老板审批驳回 |
| 通知系统 | 分级预警通知，临期、指派、争议、确稿全场景覆盖 |
| 修改溯源 | 完整记录每次修改和标注的时间轴，支持版本前后对比 |
| 团队协作 | 工作空间成员管理，支持多人实时在线协作 |

### 技术规格

| 项目 | 规格 |
|------|------|
| 支持图片格式 | JPG / JPEG / PNG / WebP / SVG / GIF |
| 单文件上限 | 50MB |
| 缩略图规格 | 320px（小图）/ 800px（中图）/ 原图 |
| 传输加密 | HTTPS |
| 身份认证 | JWT（有效期 7 天） |
| 密码存储 | bcrypt 12 轮哈希 |
| 速率限制 | 登录 30次/15min，全局 1000次/15min |
| 实时协作 | WebSocket 标注同步 |
| 日志系统 | 结构化 JSON 日志，四级分层，请求追踪 |

## 2. 快速入门

### 2.1 注册账号

1. 打开系统地址，进入登录页面
2. 点击底部「还没有账号？立即注册」
3. 填写信息：
   - **姓名**：真实姓名或花名（必填）
   - **邮箱**：有效邮箱地址（必填）
   - **密码**：至少 6 个字符（必填）
   - **确认密码**：再次输入确认（必填）
4. 系统自动校验邮箱格式、密码一致性和长度
5. 点击「注册」提交，成功后自动跳转登录页

### 2.2 登录系统

#### 邮箱密码登录

1. 输入邮箱地址和密码
2. 点击密码框右侧眼睛图标可切换密码可见性
3. 点击「登录」按钮
4. 登录成功后跳转到战情室页面

#### 微信扫码登录（演示模式）

系统支持微信 OAuth 扫码登录。未配置微信开放平台时使用演示模式。

#### 登录过期预警

JWT 过期前 5 分钟，顶部滑出黄色提醒条：「登录即将过期，请保存当前工作。[立即刷新登录]」。点击可续期登录。

### 2.3 界面概览

登录后进入系统主界面，由以下区域组成：

- **左侧导航栏**：包含「战情室」「我的待办」「项目看板」「工作空间」等入口，支持折叠/展开
- **顶部栏**：面包屑导航、通知铃铛（未读计数）、用户头像和退出按钮
- **内容区域**：根据当前页面显示对应功能模块

### 2.4 系统访问路径总览

| 路径 | 页面 | 权限 |
|------|------|------|
| `/` | 战情室（首页） | 登录用户 |
| `/projects` | 项目看板 | 登录用户 |
| `/my-tasks` | 我的待办 | 登录用户 |
| `/workspace` | 工作空间设置 | 登录用户 |
| `/project/:id` | 项目详情 | 登录用户 |
| `/project/:id/timeline` | 时间轴 | 登录用户 |
| `/project/:id/color-check` | 色差巡检 | 登录用户 |
| `/project/:id/consistency-check` | 光影一致性 | 登录用户 |
| `/project/:id/portfolio` | 作品集编辑 | 登录用户 |
| `/portfolio/:id` | 作品集预览 | 登录用户 |
| `/client/project/:shareToken` | 客户端确稿 | 免登录 |
| `/client/assets` | 客户端资产 | 免登录 |

## 3. 战情室（老板工作台）

战情室是老板和项目总监的每日工作起始页，登录后默认进入。集中展示所有项目的健康状态、成员负载和待处理事项。

### 3.1 统计面板

页面顶部展示五个核心统计卡片：

| 统计项 | 说明 |
|--------|------|
| 全部项目 | 当前空间内的项目总数 |
| 进行中 | 状态为「修改中」和「待确稿」的项目数 |
| 红灯项目 | 存在争议或严重超期的项目数 |
| 黄灯项目 | 临期预警或存在轻度问题的项目数 |
| 待处理意见 | 全部未解决意见卡片数量 |

### 3.2 项目健康度

项目列表以卡片形式展示每个项目的健康状态：

- **绿灯** 🟢：项目进展正常，无预警
- **黄灯** 🟡：有未读意见超过 24 小时，需要关注
- **红灯** 🔴：存在争议卡片或临近截止日期，需要立即处理

每个项目卡片显示：项目名称、客户名称、当前状态、待处理问题数、预警原因标签、截止日期。

### 3.3 健康度规则说明

点击战情室顶部「健康度规则」问号图标，弹出浮窗展示触发条件：

- **红灯**：有争议卡片（被反复重开 ≥2 次）、项目临期（距截止日期不足预警小时数）
- **黄灯**：有未读意见超过 24 小时

### 3.4 成员负载

页面底部展示团队成员工作负载：

- 每个成员的姓名和头像
- 当前任务数（被指派的卡片数量）/ 建议上限
- 负载进度条（绿色=正常，黄色=繁忙，红色=超负荷）
- 建议上限由老板在空间设置中自定义，默认 15

### 3.5 一键催稿

在红灯/黄灯项目卡片上，提供「提醒客户」按钮：

- 点击后系统自动向客户发送预置提醒通知
- 卡片上显示「已催稿」标签和最近催稿时间
- 催稿计数自动累加

## 4. 修图师待办列表

点击左侧「我的待办」进入，是修图师的个人工作台。

### 4.1 任务优先级

待办列表按优先级自动排序：

| 优先级 | 标识 | 说明 |
|--------|------|------|
| 争议 | 🔴 红色标记 | 被反复重开的卡片 |
| 临期 | 🟡 黄色标记 | 所属项目临近截止日期 |
| 普通 | ⚪ 默认 | 常规待处理卡片 |

每条待办显示：关联图片缩略图、项目名称和产品单元、标注内容摘要、指派时间、优先级标签。

### 4.2 快速标记已解决

1. 点击待办卡片上的「标记为已解决」
2. 卡片从待办列表移除
3. 关联项目中的意见卡片状态同步更新
4. 指派者收到完成通知

### 4.3 快速流转模式

标记「已解决」后，系统自动将下一条待办卡片高亮并滚动至视图中央，修图师可连续操作无需手动切换。

### 4.4 任务预览浮窗

鼠标悬停或点击待办卡片缩略图，弹出 280×210px 放大预览浮窗，居中展示标注区域。

### 4.5 预计耗时

意见卡片指派时可设置预计耗时（15分钟 / 30分钟 / 1小时），修图师可按预计耗时排序。

## 5. 工作空间管理

点击左侧「工作空间」进入管理页面。

### 5.1 空间基本信息

- **空间名称**：点击名称区域可直接编辑
- **Logo 上传**：支持 JPG / PNG / SVG 格式
- **套餐类型**：显示当前套餐（免费版 / 专业版 / 企业版）
- **创建时间**：工作空间的创建日期

### 5.2 成员管理

- **查看成员列表**：姓名、邮箱、角色、状态
- **邀请成员**：输入邮箱地址发送邀请
- **角色说明**：
  - **拥有者（Owner）**：全部管理权限，包括删除空间、管理成员、驳回确稿
  - **编辑者（Editor）**：项目创建、编辑、标注，不能管理空间设置
  - **查看者（Viewer）/ 外部协作者**：仅可访问被明确指派的项目，无法浏览空间内其他内容
- **移除成员**：将成员移出工作空间
- **禁用成员**：禁用后无法登录和访问空间

### 5.3 API 密钥管理

查看和复制当前空间的 API 密钥，用于第三方系统集成。

### 5.4 成员负载上限设置

在工作空间设置中可配置成员负载上限，默认 15。修改后战情室成员负载显示同步更新。

### 5.5 预设短语管理

为文字标注工具设置常用短语列表（如"调亮""压暗""去瑕疵"），标注时可通过下拉菜单快速选择。

### 5.6 快捷键配置

自定义工作空间级别的快捷键配置，支持复制、删除、下一张卡片、跳转页码等常用操作。

## 6. 项目看板

点击左侧「项目看板」进入，是日常项目管理的核心页面。

### 6.1 统计概览

看板顶部显示四个统计卡片：

| 统计项 | 说明 |
|--------|------|
| 全部项目 | 当前空间内的项目总数 |
| 进行中 | 「修改中」和「待确稿」的项目数 |
| 待处理 | 「待确认」和「草稿」的项目数 |
| 已完成 | 「已完成」和「已归档」的项目数 |

### 6.2 创建项目

1. 点击「新建项目」按钮
2. 填写信息：
   - **项目名称**（必填）
   - **客户名称**（选填）
   - **截止日期**（选填）
   - **备注**（选填）
3. 点击「创建」

### 6.3 基于模板创建项目

1. 在创建项目时选择「从模板创建」
2. 选择预置模板或自定义模板
3. 系统自动应用模板配置（产品单元、标注规范等）
4. 确认后一键创建完整项目

> 详情参见 [第14章 项目流程模板](#14-项目流程模板)

### 6.4 上传图片

1. 进入项目详情页，通过上传区域添加图片
2. 支持同时选择多个文件批量上传（最多 20 张）
3. 支持格式：JPG、JPEG、PNG、WebP、SVG、GIF
4. 单文件最大 50MB
5. 上传完成后自动生成 320px / 800px 两级缩略图

### 6.5 智能批量筛选

1. 输入行话关键词（如"去雾""过曝""背景不纯"）
2. 系统自动匹配相关图片，显示匹配结果
3. 查看 AI 给出的筛选建议
4. 点击「一键应用」将筛选结果应用到当前视图

**支持的行话模板**：去雾、过曝/欠曝、背景不纯、色温偏暖/偏冷、对比度不够等。

### 6.6 行话模板库

系统预置了常用修图术语模板，点击模板标签即可快速填入筛选关键词。

### 6.7 个人行话模板

用户可将筛选关键词保存为个人模板：

- 仅本人可见，支持增删改
- 保存后可一键填充到筛选框
- 提升日常筛选效率

### 6.8 项目列表与筛选

项目以卡片网格形式展示，每个卡片包含：

- 项目缩略图（或首字母占位图）
- 项目名称、客户名称
- 待处理数量
- 更新时间

**搜索**：支持按项目名称和客户名称模糊搜索。

**状态筛选**：点击状态标签快速筛选（全部、草稿、待确认、修改中、待确稿、已完成、已归档）。

### 6.9 仅显示未处理开关

开启后依据图片是否有关联的「未解决」意见卡片过滤，关闭后恢复显示全部图片。

### 6.10 项目列表排序

支持按更新时间、截止日期、创建时间、客户名称排序，排序偏好自动保持。

### 6.11 筛选状态提示条

筛选激活时，列表顶部显示蓝色提示条：「正在按「XX」筛选，共 N 个项目。[清除筛选]」

### 6.12 项目状态流转

项目共有 6 种状态：

```
草稿 → 待确认 → 修改中 → 待确稿 → 已完成 → 已归档
```

| 状态 | 含义 | 典型操作 |
|------|------|---------|
| 草稿 | 项目刚创建，尚未开始 | 上传图片、编辑项目信息 |
| 待确认 | 已提交给客户审阅 | 等待客户提交意见 |
| 修改中 | 客户已提出修改意见 | 根据意见标注进行修改 |
| 待确稿 | 修改完成，等待客户确认 | 客户确认或驳回 |
| 已完成 | 客户已确稿，交付完成 | 导出作品集、下载成片 |
| 已归档 | 项目已归档，不再活跃 | 查看历史记录 |

## 7. 项目详情与图片标注

点击项目卡片进入项目详情页，这是最核心的工作页面。页面分为三个主要区域：顶部工具栏、左侧图片查看器、右侧意见卡片面板。

### 7.1 顶部工具栏

| 按钮 | 功能 |
|------|------|
| ← 返回 | 返回项目看板 |
| 分享 | 生成客户端分享链接，设置有效期（永久/7天/30天） |
| 色差巡检 | 对项目中所有图片进行色差分析 |
| 光影巡检 | 对项目进行跨场景光影一致性分析 |
| 导出意见 | 将全部意见卡片导出为 PDF |
| 作品集 | 进入作品集编辑页面 |
| 时间轴 | 查看项目的完整修改溯源时间轴 |
| 抽查 | 进入抽查模式，连续审查已解决卡片 |

### 7.2 图片查看器

图片查看器位于页面左侧，是主要的图片浏览和操作区域。

**基础操作**：

- **翻页浏览**：点击箭头按钮或键盘方向键 ← →
- **全屏模式**：点击全屏按钮，按 ESC 退出
- **缩放**：鼠标滚轮或双指手势缩放
- **平移**：缩放后拖拽移动
- **旋转**：点击旋转按钮，每点击一次旋转 90 度（仅影响浏览视图，不修改原图）
- **长图指示器**：超长图片显示位置指示器

**状态提示**：

- 加载中：显示加载动画和模糊占位图
- 加载失败：显示错误提示和重试按钮

### 7.3 标注工具

标注工具是核心的协作功能，允许在图片上精确标记修改意见。

**六种标注工具**：

| 工具 | 用途 | 颜色 | 线宽 |
|------|------|------|------|
| 画笔 | 自由绘制线条 | 红/黄/蓝/白 | 3/5/10 |
| 箭头 | 精确指向位置 | 红/黄/蓝/白 | 3/5 |
| 矩形 | 矩形框选区域 | 红/黄/蓝/白 | 3/5/10 |
| 椭圆 | 椭圆圈选区域 | 红/黄/蓝/白 | 3/5/10 |
| 文字 | 直接添加文字说明 | 红/黄/蓝/白 | 字号 14/18/24 |
| 橡皮擦 | 擦除已有标注 | - | - |

**标注特点**：

- 使用相对坐标系统（左上角 0,0，右下角 1,1），缩放不影响标注位置
- 标注数据自动保存，实时同步到服务器
- 支持多人协作标注

**使用步骤**：

1. 选择标注工具（如画笔）
2. 选择颜色和线宽
3. 在图片上拖动鼠标绘制标注
4. 标注完成后自动创建对应的意见卡片
5. 在右侧面板的意见卡片中填写具体修改说明

### 7.4 快捷键

| 快捷键 | 功能 |
|--------|------|
| **A** | 切换标注工具（画笔 ↔ 箭头） |

### 7.6 意见卡片与任务指派

意见卡片与标注一一对应，记录修改意见和沟通内容。

**卡片内容**：

- 关联的标注缩略图
- 修改意见文字描述
- 卡片状态：未解决 / 已解决
- 指派人信息（头像 + 姓名）
- 创建时间
- 最后编辑信息（「最后由 [姓名] 编辑于 [时间]」）

- 争议卡片红色高亮显示
- 自动推送通知给项目负责人
- 战情室中对应项目标记为红灯

**卡片操作**：

- **添加意见**：标注完成后输入修改意见
- **解决/重开**：修改完成后点击「标记为已解决」；如需重新讨论点击「重开」
- **编辑**：支持编辑已创建的意见卡片内容
- **排序**：卡片按创建顺序排列

## 8. 色差巡检

色差巡检功能用于自动检测一组图片中存在的色温、亮度和对比度偏差，帮助保证成片色彩一致性。

### 8.1 一键巡检

1. 在项目详情页顶部工具栏，点击「色差巡检」按钮
2. 系统开始对项目中所有图片进行逐张分析
3. 进度条实时显示检测进度
4. 检测完成后自动跳转到巡检结果页面

### 8.2 场景化巡检修正

针对不同拍摄场景提供定制化巡检方案：

| 场景预设 | 适用场景 | 关注重点 |
|---------|---------|---------|
| 电商白底图 | 电商产品拍摄 | 背景纯度、色温一致性 |
| 美食暖调 | 美食摄影 | 暖色调一致性、饱和度 |
| 人像肤质 | 人像摄影 | 肤色还原、曝光均匀度 |
| 通用 | 通用场景 | 全维度色差检测 |

**AI 修正预览**：查看修正前后叠加对比，拖动透明度滑块查看差异，确认后点击「应用修正」。

### 8.3 巡检结果查看

巡检结果展示：

- 总图片数、异常数量
- 异常列表：缩略图、偏差类型（色温/亮度/对比度）、偏差值、修改建议

### 8.4 导出报告

点击「导出 PDF」按钮，将巡检结果导出为 PDF 报告。

### 8.5 局部放大镜对比

AI 修正预览的叠加对比界面中，异常区域旁自动出现 200% 放大镜视图，方便精确对比色差细节。

### 8.6 批量/单选混合应用

巡检结果列表增加复选框：

- 支持部分选中后「应用修正于选中项」
- 保留「一键应用全部」按钮
- 未被选中图片不受影响

## 9. 光影一致性巡检

> BETA 功能

光影一致性巡检（光影探照灯）是高级品控工具，超越单张图片的色差分析，专注于同场景组图的光影逻辑一致性。

### 9.1 一键巡检

1. 在项目详情页顶部工具栏，点击「光影巡检」按钮
2. 进入光影一致性巡检页面
3. 点击「一键巡检」开始分析
4. 进度条实时显示分析进度
5. 完成后展示完整巡检报告

### 9.2 场景组分析

系统按产品单元将图片分组，分析每组的光影特征：

- **主光源方向**：检测每组图片的主光源来向
- **一致性判定**：标记每组为「一致」或「不一致」
- **一致性评分**：0-100 分，卡片颜色渐变（绿→黄→红）

### 9.3 异常报告

检测 6 类光影异常：

| 异常类型 | 严重度 | 说明 |
|---------|--------|------|
| 光源方向 | 高 | 主光源方向不一致，疑似灯位偏移 |
| 高光位置 | 高 | 面部高光位置与组图其他图片不一致 |
| 阴影角度 | 中 | 阴影角度偏差超过阈值 |
| 色温 | 中 | 同组图片色温不统一 |
| 曝光 | 低 | 曝光量差异明显 |
| 反光 | 低 | 反光面亮度差异过大 |

每条异常包含：严重度标签、异常描述、异常图片与正常参考图片的并排对比、修复建议。

### 9.4 导出报告

点击「导出报告」按钮，支持打印或导出 PDF 格式。

### 9.5 BETA 安全声明

巡检入口和报告顶部显示：「BETA 功能：巡检结果仅供参考，不会自动修改任何图片。」

### 9.6 忽略异常与恢复

每条异常报告右侧有「忽略」按钮：

- 点击后该异常置灰折叠
- 可在「已忽略列表」中恢复
- 忽略后不影响评分

## 11. 客户端分享与确稿

客户端分享功能允许客户在**不注册账号**的情况下查看项目进度并提交确稿。

### 11.1 生成分享链接

1. 在项目详情页顶部工具栏，点击「分享」按钮
2. 设置有效期：
   - **永久有效**
   - **7 天有效**
   - **30 天有效**
3. 点击「生成链接」复制分享 URL
4. 将链接发送给客户

### 11.2 客户端查看项目

客户通过分享链接进入后看到：

- 项目名称和状态
- 进度条展示整体完成进度
- 图片查看器（支持翻页浏览）
- 倒计时/预计完成时间
- 意见卡片面板
- 确稿确认按钮

### 11.3 首次打开气泡引导

新客户首次打开分享链接时，出现 3 个轻量气泡引导：

- 「双指缩放查看细节」
- 「点击画笔开始标注」
- 「写完意见点发送」
- 点击可跳过，关闭后不再显示

### 11.4 保护型确稿流程

确稿后项目进入只读状态，防止客户随意修改需求。

**确稿步骤**：

1. 客户浏览所有图片和修改意见
2. 确认所有修改已按要求完成
3. 点击「确稿」按钮
4. 系统弹出对比滑块确稿弹窗，展示修改前后对比
5. 二次确稿弹窗提醒：「确稿后如需修改需联系摄影师申请。确认提交吗？」
6. 确认后项目状态变更为「已完成」

**确稿对比高亮**：修改过的区域用半透明虚线框高亮，附简要文字说明（如「曝光+0.5」）。

**申请修改**：

1. 确稿后如需修改，客户点击「申请修改」
2. 填写修改原因（**必须不少于 5 个字**）
3. 系统发送通知给项目负责人
4. 只有老板角色可以驳回确稿，重新开放编辑

**驳回确稿**：

1. 老板在项目详情页看到修改申请
2. 审核修改原因
3. 点击「驳回确稿」，项目状态恢复为「修改中」
4. 团队成员可以继续编辑
5. 客户收到驳回通知

### 11.5 意见已读回执

客户浏览意见卡片及关联标注区域超 3 秒，自动标记「对方已读」。工作室端显示「✓ 对方已读」及时间。

## 18. 常见问题

### Q1：忘记密码怎么办？

请联系工作空间管理员重置密码。系统暂不支持自助重置密码。

### Q2：支持哪些图片格式？文件大小限制？

支持 JPG、JPEG、PNG、WebP、SVG、GIF。单文件最大 50MB。

### Q3：如何邀请团队成员？

进入「工作空间」→ 成员管理 → 点击「邀请成员」→ 输入邮箱地址。

### Q4：项目状态如何变更？

项目状态会随着操作自动流转。上传图片后可从「草稿」手动变更为「待确认」；客户提交意见后自动变为「修改中」；修改完成后可手动变更为「待确稿」；客户确稿后自动变为「已完成」。

### Q5：确稿后还能修改吗？

确稿后项目进入只读状态。客户需点击「申请修改」提交不少于 5 个字的原因，老板审核后点击「驳回确稿」，项目恢复为「修改中」。

### Q6：分享链接失效了怎么办？

在项目详情页重新生成分享链接即可。建议根据项目周期设置合适的有效期。

### Q7：标注会丢失吗？

不会。标注使用相对坐标系统存储，所有标注自动保存到服务器，不会因浏览器刷新或关闭而丢失。

### Q8：如何查看谁做了哪些修改？

进入项目详情页 → 点击「时间轴」→ 查看完整修改溯源记录。

### Q9：什么是争议预警？

当一张意见卡片被反复重开 ≥2 次时，系统自动标记为「争议」。争议卡片红色高亮，推送通知给项目负责人，战情室中对应项目显示红灯。

### Q10：作品集可以设置密码访问吗？

当前版本暂不支持密码访问，作品集发布后为公开访问。建议使用分享链接有效期功能控制访问。

### Q11：浏览器兼容性如何？

推荐使用最新版 Chrome、Edge、Firefox 或 Safari。不支持 IE。

### Q12：数据安全如何保障？

- HTTPS 加密传输
- bcrypt 12 轮哈希存储密码
- JWT Token 身份认证
- 速率限制防暴力破解
- 参数化查询防 SQL 注入
- 输入校验防 XSS 攻击
- 生产环境强制 HSTS
