## 1. 产品概述

**易拍选（EpicShot）** 是一款面向商业摄影行业的在线协作平台，帮助摄影师、修图师与客户之间高效完成图片审阅、标注修改、色差巡检和成片交付。

### 核心功能

| 功能模块 | 说明 |
|---------|------|
| 战情室 | 老板每日工作起始页，项目健康度一目了然，成员负载实时监控 |
| 项目管理 | 创建和管理摄影项目，跟踪项目进度，支持流程模板一键创建 |
| 图片标注 | 使用画笔、箭头、矩形、椭圆、文字等工具在图片上精确标注修改意见 |
| 任务指派 | 将意见卡片指派给特定修图师，争议预警自动标记反复重开的问题 |
| 色差巡检 | 一键检测全组图片的色温、亮度和对比度偏差，支持场景化修正 |
| 光影一致性巡检 | 跨场景分析光影逻辑一致性，检测灯位偏移、高光不一致等问题 |
| 智能批量筛选 | 输入行话关键词，AI 自动匹配相关图片，一键应用筛选建议 |
| 作品集 | AI 辅助生成精美作品集，支持手机框架预览和公开展示 |
| 客户端分享 | 生成免登录分享链接，客户可直接查看项目并确认确稿 |
| 保护型确稿 | 确稿后锁定编辑，客户需申请修改由老板审批驳回 |
| 通知系统 | 分级预警通知，临期、指派、争议、确稿全场景覆盖 |
| 修改溯源 | 完整记录每次修改和标注的时间轴，支持版本前后对比 |
| 团队协作 | 工作空间成员管理，支持多人实时在线协作 |

### 技术亮点

- 支持 JPG / PNG / WebP / SVG 等多种图片格式
- 单文件最大支持 50MB
- 自动生成多级缩略图（320px / 800px / 原图），优化加载速度
- 支持 HTTPS 加密传输
- 基于 JWT 的身份认证，保障数据安全
- WebSocket 实时协作（标注同步、光标位置共享）
- 结构化 JSON 日志，四级分层，支持请求追踪

## 2. 快速入门

### 2.1 注册账号

1. 在浏览器中打开系统地址，进入登录页面
2. 点击页面底部的「还没有账号？立即注册」链接，跳转到注册页面
3. 填写以下信息：
   - **姓名**：您的真实姓名或花名（必填）
   - **邮箱**：用于登录和接收通知的有效邮箱地址（必填）
   - **密码**：至少 6 个字符（必填）
   - **确认密码**：再次输入密码以确认（必填）
4. 系统会自动校验：
   - 邮箱格式是否正确
   - 两次输入的密码是否一致
   - 密码长度是否满足最低要求（≥6 位）
5. 点击「注册」按钮提交
6. 注册成功后，系统会自动跳转到登录页面

### 2.2 登录系统

#### 邮箱密码登录

1. 在登录页面，输入您的**邮箱地址**和**密码**
2. 点击密码输入框右侧的眼睛图标，可以切换密码的显示/隐藏
3. 点击「登录」按钮
4. 登录成功后，系统会跳转到战情室页面

#### 忘记密码

如果忘记密码，请点击登录页面的「忘记密码？」链接，系统会提示您联系管理员重置密码。

### 2.3 界面概览

登录成功后，您将看到系统主界面，由以下区域组成：

- **左侧导航栏**：包含「战情室」「我的待办」「项目看板」「工作空间」等入口，底部显示当前套餐类型
- **顶部栏**：显示面包屑导航、通知铃铛（未读计数）、用户头像和退出按钮
- **内容区域**：根据当前页面显示对应的功能模块

左侧导航栏支持**折叠/展开**，点击侧边栏顶部的 ◀/▶ 按钮即可切换，折叠后可获得更大的内容显示空间。

## 3. 战情室（老板工作台）

战情室是老板和项目总监的每日工作起始页，登录后默认进入。集中展示所有项目的健康状态、成员负载和待处理事项。

### 3.1 统计面板

页面顶部展示五个核心统计卡片：

| 统计项 | 说明 |
|--------|------|
| 全部项目 | 当前空间内的项目总数 |
| 进行中 | 状态为「修改中」和「待确稿」的项目数 |
| 红灯项目 | 存在争议或严重超期的项目数 |
| 黄灯项目 | 临期预警或存在轻度问题的项目数 |
| 待处理 | 需要老板关注和决策的事项数 |

### 3.2 项目健康度

项目列表以卡片形式展示每个项目的健康状态：

- **绿灯** 🟢：项目进展正常，无预警
- **黄灯** 🟡：临近截止日期或存在轻微问题，需要关注
- **红灯** 🔴：存在争议卡片、严重超期，需要立即处理

每个项目卡片显示：
- 项目名称和客户名称
- 当前状态标签
- 待处理问题数
- 预警原因标签（如"临期""争议"）
- 截止日期

### 3.3 成员负载

页面底部展示团队成员的工作负载：

- 每个成员的**姓名和头像**
- **当前任务数**（被指派的卡片数量）
- **负载进度条**（绿色=正常，黄色=繁忙，红色=超负荷）

## 4. 修图师待办列表

点击左侧导航栏的「我的待办」进入，是修图师的个人工作台。

### 4.1 任务优先级

待办列表按优先级自动排序：

| 优先级 | 标识 | 说明 |
|--------|------|------|
| 争议 | 🔴 红色标记 | 被反复重开的卡片，需要重点关注 |
| 临期 | 🟡 黄色标记 | 所属项目临近截止日期 |
| 普通 | ⚪ 默认 | 常规待处理卡片 |

每条待办显示：
- 关联图片缩略图
- 项目名称和产品单元
- 标注内容摘要
- 指派时间
- 优先级标签

### 4.2 快速标记已解决

1. 在待办卡片上点击「标记为已解决」
2. 卡片从待办列表移除
3. 关联项目中的意见卡片状态同步更新
4. 指派者收到完成通知

## 5. 工作空间管理

工作空间是您团队协作的基本单位。点击左侧导航栏的「工作空间」进入管理页面。

### 5.1 空间基本信息

- **空间名称**：点击名称区域可直接编辑，修改后即时生效
- **Logo 上传**：点击 Logo 区域可上传自定义品牌标识，支持 JPG / PNG / SVG 格式
- **套餐类型**：显示当前套餐（免费版 / 专业版 / 企业版）
- **创建时间**：显示工作空间的创建日期

### 5.2 成员管理

在工作空间页面的「成员管理」区域，您可以：

- **查看成员列表**：显示所有成员的姓名、邮箱、角色（拥有者/编辑者）和状态（活跃/已禁用）
- **邀请成员**：点击「邀请成员」按钮，输入被邀请人的邮箱地址，系统会发送邀请通知
- **修改角色**：点击成员行中的角色标签，可在「拥有者」和「编辑者」之间切换
  - **拥有者**：拥有空间的所有管理权限，包括删除空间、管理成员等
  - **编辑者**：可以进行项目创建、编辑、标注等操作，但不能管理空间设置
- **移除成员**：点击成员行中的「移除」按钮，可将该成员移出工作空间
- **禁用成员**：可将成员状态设为「已禁用」，禁用后该成员无法登录和访问空间

### 5.3 API 密钥管理

> 此功能面向开发人员，用于通过 API 对接外部系统。

在工作空间页面可查看和复制当前空间的 API 密钥，用于第三方系统集成。

## 6. 项目看板

项目看板是日常项目管理的核心页面。点击左侧导航栏的「项目看板」进入。

### 6.1 统计概览

看板顶部显示四个统计卡片：

| 统计项 | 说明 |
|--------|------|
| 全部项目 | 当前空间内的项目总数 |
| 进行中 | 状态为「修改中」和「待确稿」的项目数 |
| 待处理 | 状态为「待确认」和「草稿」的项目数 |
| 已完成 | 状态为「已完成」和「已归档」的项目数 |

### 6.2 创建项目

1. 点击看板右上角的「新建项目」按钮
2. 在弹出的对话框中填写项目信息：
   - **项目名称**：为项目起一个易于识别的名称（必填）
   - **客户名称**：客户的姓名或公司名称（选填）
   - **截止日期**：项目的期望完成日期（选填）
   - **备注**：项目的补充说明（选填）
3. 点击「创建」按钮，新项目将出现在项目列表中

### 6.3 基于模板创建项目

> 详情参见 [第14章 项目流程模板](#14-项目流程模板)

1. 在创建项目时，可选择「从模板创建」
2. 选择预置模板或自定义模板
3. 系统自动应用模板中的配置（产品单元、标注规范等）
4. 确认后一键创建完整项目

### 6.4 上传图片

创建项目后，您可以通过以下方式上传图片：

1. **在项目详情页上传**：进入项目详情页后，通过上传区域添加图片
2. **批量上传**：支持同时选择多个文件批量上传

**支持的图片格式**：JPG、JPEG、PNG、WebP、SVG、GIF

**上传限制**：单文件最大 50MB

上传完成后，系统会自动生成多级缩略图（小图 320px、中图 800px），在不同场景下自动加载合适尺寸，优化浏览速度。

### 6.5 智能批量筛选

智能批量筛选功能帮助您快速找到需要处理的图片。

**使用步骤**：
1. 进入项目详情页，找到批量筛选面板
2. 在输入框中输入行话关键词（如"去雾""过曝""背景不纯"）
3. 系统自动匹配相关图片，显示匹配结果网格
4. 查看 AI 给出的筛选建议
5. 点击「一键应用」将筛选结果应用到当前视图

**支持的行话模板**：
- 去雾 — 匹配需要去雾处理的图片
- 过曝/欠曝 — 匹配曝光问题的图片
- 背景不纯 — 匹配背景需要处理的图片
- 色温偏暖/偏冷 — 匹配白平衡问题的图片
- 对比度不够 — 匹配对比度需要增强的图片

### 6.6 行话模板库

系统预置了常用修图术语模板，方便快速调用：

1. 在批量筛选面板中，查看「行话模板」区域
2. 点击模板标签即可快速填入筛选关键词
3. 支持自定义行话模板（未来版本）

### 6.7 项目列表与筛选

项目以卡片网格形式展示，每个卡片包含：
- 项目缩略图（或首字母占位图）
- 项目名称
- 客户名称
- 待处理数量
- 更新时间

**搜索功能**：看板顶部的搜索框支持按项目名称和客户名称进行模糊搜索。

**状态筛选**：点击搜索框下方的状态标签（全部、草稿、待确认、修改中、待确稿、已完成、已归档），可以快速筛选特定状态的项目。

### 6.8 项目状态说明

项目共有 6 种状态，流转路径如下：

```
草稿 → 待确认 → 修改中 → 待确稿 → 已完成 → 已归档
```

| 状态 | 含义 | 典型操作 |
|------|------|---------|
| 草稿 | 项目刚创建，尚未开始 | 上传图片、编辑项目信息 |
| 待确认 | 已提交给客户审阅，等待反馈 | 等待客户提交意见 |
| 修改中 | 客户已提出修改意见，团队正在处理 | 根据意见标注进行修改 |
| 待确稿 | 修改完成，等待客户最终确认 | 客户确认或驳回 |
| 已完成 | 客户已确稿，项目交付完成 | 导出作品集、下载成片 |
| 已归档 | 项目已归档，不再活跃 | 查看历史记录 |

## 7. 项目详情与标注

点击项目卡片进入项目详情页，这是最核心的工作页面。页面分为三个主要区域。

### 7.1 顶部工具栏

工具栏位于详情页顶部，提供以下功能按钮：

| 按钮 | 功能 |
|------|------|
| ← 返回 | 返回项目看板 |
| 分享 | 生成客户端分享链接，设置有效期（永久/7天/30天） |
| 色差巡检 | 对项目中所有图片进行色差分析 |
| 光影巡检 | 对项目进行跨场景光影一致性分析 |
| 导出意见 | 将全部意见卡片导出为 PDF 文档 |
| 作品集 | 进入作品集编辑页面 |
| 时间轴 | 查看项目的完整修改溯源时间轴 |

### 7.2 图片查看器

图片查看器位于页面左侧，是主要的图片浏览和操作区域。

**基础操作**：

- **翻页浏览**：点击图片两侧的箭头按钮，或使用键盘方向键切换图片
- **全屏模式**：点击全屏按钮进入全屏查看，按 ESC 或再次点击退出
- **缩放**：使用鼠标滚轮或双指手势缩放图片
- **平移**：缩放后拖拽图片移动查看不同区域
- **旋转**：点击旋转按钮，每点击一次旋转 90 度
- **长图指示器**：对于超长图片，查看器会显示位置指示器，帮助您了解当前可视区域在整张图中的位置

**状态提示**：
- 图片加载中：显示加载动画
- 图片加载失败：显示错误提示和重试按钮

### 7.3 标注工具

标注工具是核心的协作功能，允许您在图片上精确标记修改意见。

**工具栏切换**：在图片查看器底部或侧边可切换标注工具。

**支持的标注工具**：

| 工具 | 图标示意 | 用途 | 可配置项 |
|------|---------|------|---------|
| 画笔 | 自由绘制 | 圈出需要修改的区域，画自由线条 | 颜色、线宽（3/5/10） |
| 箭头 | 指向标记 | 精确指向某个需要修改的位置 | 颜色、线宽（3/5） |
| 矩形 | 矩形框选 | 框选一个区域，如裁剪范围 | 颜色、线宽 |
| 椭圆 | 椭圆圈选 | 圈选圆形区域，如面部修饰 | 颜色、线宽 |
| 文字 | 文字标注 | 在图片上直接添加文字说明 | 颜色、字号（14/18/24） |
| 橡皮擦 | 擦除标注 | 擦除已有的标注线条 | - |

**颜色选择**：支持 4 种标注颜色 — 红色（#FF0000）、黄色（#FFCC00）、蓝色（#0066FF）、白色（#FFFFFF）。

**标注特点**：
- 所有标注使用**相对坐标系统**（左上角为 0,0，右下角为 1,1），无论图片如何缩放，标注位置始终准确
- 标注数据自动保存，实时同步到服务器
- 支持多人协作标注（WebSocket 实时同步）

**使用步骤**：
1. 选择标注工具（如画笔）
2. 选择颜色和线宽
3. 在图片上拖动鼠标绘制标注
4. 标注完成后，系统会自动创建对应的意见卡片
5. 在右侧面板的意见卡片中填写具体的修改说明

### 7.4 快捷键

图片查看器支持以下键盘快捷键，提升专业用户操作效率：

| 快捷键 | 功能 |
|--------|------|
| **A** | 切换标注工具（画笔 ↔ 箭头） |
| **空格（按住）** | 临时切换到抓手工具，拖拽移动画布 |
| ← → | 上一张 / 下一张图片 |
| ESC | 退出全屏模式 |

### 7.5 产品单元管理

产品单元用于对图片进行分组管理，方便按产品类别组织拍摄内容。

- **创建产品单元**：点击「新产品单元」按钮，输入产品名称
- **拖拽排序**：产品单元支持拖拽调整顺序
- **图片归类**：将图片拖拽到对应的产品单元下
- **展开/折叠**：点击产品单元名称可展开或折叠其下的图片列表

### 7.6 意见卡片与任务指派

意见卡片与标注一一对应，用于记录具体的修改意见和沟通内容。

**卡片内容**：
- 关联的标注缩略图
- 修改意见文字描述
- 卡片状态：未解决 / 已解决
- 指派人信息（头像 + 姓名）
- 创建时间
- 争议标记（反复重开 ≥2 次显示红色标记）

**任务指派**：
1. 在意见卡片上，点击指派区域
2. 选择要指派的修图师
3. 被指派的修图师将收到通知
4. 卡片出现在该修图师的「我的待办」列表中

**争议预警**：
- 当一张卡片被反复重开 ≥2 次，系统自动标记为「争议」
- 争议卡片显示红色高亮标记
- 自动推送通知给项目负责人
- 在战情室中该项目标记为红灯

**卡片操作**：
- **添加意见**：标注完成后，在卡片中输入修改意见说明
- **解决/重开**：修改完成后，点击「标记为已解决」；如需重新讨论，点击「重开」
- **排序**：卡片按创建顺序排列

## 9. 光影一致性巡检

> V1.1 新增功能（BETA）

光影一致性巡检（光影探照灯）是高级品控工具，超越单张图片的色差分析，专注于同场景组图的光影逻辑一致性。

### 9.1 一键巡检

1. 在项目详情页顶部工具栏，点击「光影巡检」按钮
2. 进入光影一致性巡检页面
3. 点击「一键巡检」开始分析
4. 进度条实时显示分析进度
5. 完成后展示完整巡检报告

### 9.2 场景组分析

系统按产品单元将图片分组，分析每组的光影特征：

- **主光源方向**：检测每组图片的主光源来向（左上方/右上方/正上方/左前方/右前方）
- **一致性判定**：标记每组为「一致」或「不一致」
- **一致性评分**：0-100 分综合评分，卡片颜色渐变（绿→黄→红）

### 9.3 异常报告

系统检测以下 6 类光影异常：

| 异常类型 | 严重度 | 说明 |
|---------|--------|------|
| 光源方向 | 高 | 主光源方向不一致，疑似灯位偏移 |
| 高光位置 | 高 | 面部高光位置与组图其他图片不一致 |
| 阴影角度 | 中 | 阴影角度偏差超过阈值，光源高度可能变化 |
| 色温 | 中 | 同组图片色温不统一，白平衡设置可能变更 |
| 曝光 | 低 | 曝光量差异明显，闪光灯功率可能不一致 |
| 反光 | 低 | 反光面亮度差异过大，柔光设备位置可能偏移 |

每条异常包含：
- 严重度标签（高/中/低，颜色区分）
- 异常描述
- 异常图片与正常参考图片的并排对比
- 具体的修复建议

### 9.4 导出报告

点击「导出报告」按钮，支持打印或导出 PDF 格式的巡检报告。

## 11. 客户端分享与确稿

客户端分享功能允许客户在不注册账号的情况下查看项目进度并提交确稿。

### 11.1 生成分享链接

1. 在项目详情页顶部工具栏，点击「分享」按钮
2. 设置分享链接的有效期：
   - **永久有效**：链接永久有效
   - **7 天有效**：链接 7 天后自动失效
   - **30 天有效**：链接 30 天后自动失效

4. 将链接发送给客户

### 11.2 客户端查看项目

客户通过分享链接进入后，看到的页面包含：

- **项目名称和状态**：顶部显示项目名称和当前状态
- **进度条**：直观展示项目整体完成进度

### 11.3 保护型确稿流程

> V1.1 新增功能

保护型确稿防止客户在确稿后随意修改需求，确保团队工作成果不被随意推翻。

**确稿步骤**：
1. 客户浏览所有图片和修改意见
2. 确认所有修改已按要求完成
3. 点击「确稿」按钮
4. 系统弹出对比滑块确稿弹窗，展示修改前后对比
5. 客户再次确认后，项目状态变更为「已完成」
6. **确稿后项目进入只读状态**，所有编辑操作被锁定

**申请修改**：
1. 确稿后如需修改，客户点击「申请修改」
2. 填写修改原因
3. 系统发送通知给项目负责人（老板）
4. 只有老板角色可以驳回确稿，重新开放编辑

**驳回确稿**：
1. 老板在项目详情页看到修改申请
2. 审核修改原因
3. 点击「驳回确稿」，项目状态恢复为「修改中」
4. 团队成员可以继续编辑
5. 客户收到驳回通知

## 16. 常见问题

### Q1：忘记密码怎么办？

请联系您的工作空间管理员重置密码。系统暂不支持自助重置密码功能。

### Q2：支持哪些图片格式？

支持 JPG、JPEG、PNG、WebP、SVG、GIF 格式。单文件最大 50MB。

### Q3：如何邀请团队成员？

进入「工作空间」页面，在成员管理区域点击「邀请成员」，输入对方的邮箱地址即可。

### Q4：项目状态如何变更？

项目状态会随着操作自动流转：
- 上传图片后可从「草稿」手动变更为「待确认」
- 客户提交意见后自动变为「修改中」
- 修改完成后可手动变更为「待确稿」
- 客户确稿后自动变为「已完成」
- 可在项目设置中手动归档

### Q5：确稿后还能修改吗？

确稿后项目进入只读状态。如需修改：
1. 客户点击「申请修改」提交修改原因
2. 老板审核后点击「驳回确稿」
3. 项目恢复为「修改中」状态，团队可继续编辑

### Q6：分享链接失效了怎么办？

在项目详情页重新生成分享链接即可。建议根据项目周期设置合适的有效期。

### Q7：标注会丢失吗？

标注数据使用相对坐标系统存储，与图片缩放比例无关。所有标注自动保存到服务器，不会因浏览器刷新或关闭而丢失。

### Q8：如何查看谁做了哪些修改？

进入项目详情页，点击「时间轴」按钮，可查看完整的修改溯源记录，包括标注、指派、修订、争议、确稿和状态变更的完整历史。

### Q9：什么是争议预警？

当一张意见卡片被反复重开 ≥2 次时，系统自动标记为「争议」。争议卡片红色高亮显示，并推送通知给项目负责人。战情室中对应项目显示红灯。

### Q10：作品集可以设置密码访问吗？

当前版本暂不支持密码访问，作品集发布后为公开访问。如需限制访问，建议使用分享链接的有效期功能。

### Q11：浏览器兼容性如何？

推荐使用最新版 Chrome、Edge、Firefox 或 Safari 浏览器。不支持 IE 浏览器。

### Q12：数据安全如何保障？

- 所有数据传输使用 HTTPS 加密
- 用户密码使用 bcrypt 12 轮哈希存储
- 身份认证使用 JWT Token，有效期 7 天
- 生产环境强制要求配置 JWT_SECRET 密钥
- 速率限制防止暴力破解
- 所有数据库查询使用参数化查询，防止 SQL 注入
- 输入严格校验，防止 XSS 攻击
