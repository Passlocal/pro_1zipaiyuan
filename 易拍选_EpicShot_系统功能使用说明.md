## 1. 快速入门

### 1.1 系统概述

易拍选 (EpicShot) 是一个在线摄影审片协作平台，连接**品牌方客户**、**工作室老板**和**修图师**三方，实现从图片上传、标注审阅、修图改稿到成品交付的全流程在线化。

### 1.2 技术环境

- 浏览器：Chrome / Edge / Safari 最新版
- 网络：建议稳定的互联网连接
- 触屏设备：支持手机/平板触屏标注

### 1.3 登录

1. 打开平台地址
2. 输入邮箱和密码登录
3. 默认管理员账号：`zhang@epicshot.com`（仅供开发环境）

---

## 2. 角色与权限

| 角色 | 权限范围 | 核心功能 |
|------|---------|---------|
| **工作室老板 (Owner)** | 所有功能 | 项目管理、战情室、工作空间设置、成员管理、财务统计 |
| **修图师 (Editor)** | 项目相关 | 待办任务处理、图片标注、卡片管理、作品集编辑 |
| **品牌方客户 (Client)** | 分享端 | 审片确认、标注意见、AI风格预览、修改请求 |

### 2.1 角色路由规则

- 老板登录后默认进入**战情室**
- 修图师登录后默认进入**我的待办**
- 老板专属页面（战情室、工作空间）对修图师不可见

## 3. 战情室 — 全局监控

**路径**: `/`（仅老板可见）

### 3.1 功能概述

战情室是老板的全局仪表盘，实时监控所有项目的健康状况和团队成员的工作负载。

### 3.2 项目健康监控

- **健康状态**：绿色(正常) / 黄色(预警) / 红色(严重)
- **预警维度**：争议卡片数量、未解决卡片数量、临近截止日期
- **统计卡片**：项目总数、活跃项目数、红/黄项目数、未解决总数
- **项目列表**：展示每个项目的名称、客户、状态、健康度、未解决数

### 3.3 成员负载

- 展示每个成员的任务数量和任务详情
- 支持展开查看成员在各项目中的具体任务
- 支持查看成员任务是否为临期（urgent）

### 3.4 财务分析

- 合同金额统计
- 按项目/月份汇总收入
- 支付阶段汇总（V1.18 新增）

### 3.5 数据导出

- 支持导出仪表盘数据为 Excel 格式

## 4. 项目看板 — 项目管理

**路径**: `/projects`

### 4.1 功能概述

项目看板是项目的管理中心，支持创建、搜索、筛选和管理所有项目。

### 4.2 创建项目

1. 点击"创建项目"按钮
2. 填写项目信息：
   - **项目名称**（必填）
   - **客户名称**
   - **截止日期**
   - **合同金额**
   - **付款阶段**（V1.18 新增）：点击"添加付款阶段"可设置分阶段付款，如定金/中期款/尾款
3. 各阶段金额之和与合同总金额不一致时，系统会显示警告提示
4. 点击"确认创建"

### 4.3 项目列表

- 显示所有项目卡片，包含名称、客户、状态、截止日期
- 状态标签：草稿 / 待确认 / 修改中 / 待确稿 / 已完成 / 已归档
- 支持搜索（按项目名称、客户名称）
- 支持状态筛选

### 4.4 项目封面

- 点击项目卡片的封面区域可上传自定义封面图
- 支持批量生成缩略图

### 4.5 项目模板

- 可将常用项目配置保存为模板
- 从模板快速创建新项目，复用项目结构

## 5. 项目详情 — 审片协作

**路径**: `/project/:id`

### 5.1 功能概述

项目详情页是审片协作的核心工作区，支持图片查看、标注、评论、修图、快速流转等完整工作流。

### 5.2 页面布局

- **左侧**：图片查看器（支持标注）
- **右侧**：产品单元树 + 意见卡片列表
- **顶部**：工具栏（分享、下载、交付包、操作日志等）

### 5.3 产品单元管理

- 每个项目可创建多个产品单元（如 SKU、场景分类）
- 支持拖拽排序
- 支持树形结构展示
- 每个单元下可上传多张图片

### 5.4 图片管理

- 上传图片到指定产品单元
- 支持批量上传
- 查看图片原图和缩略图
- 图片切换（点击缩略图或使用键盘快捷键）

### 5.5 意见卡片管理

- 创建图片标注意见卡片
- 卡片状态：未解决 / 已解决
- 支持指派卡片给特定修图师
- 支持争议标记（对修图结果不满意可标记争议）
- 争议超过阈值自动预警
- 快速流转模式：提交当前卡片后自动切换到下一张

### 5.6 快速流转模式

- 逐张处理图片意见卡片
- 提交并自动跳转下一张（V1.18 图片预加载，无白屏闪烁）
- 支持提交确认开关（提交前二次确认）
- 键盘快捷键支持（Ctrl+Enter 提交）

### 5.7 讨论区

- 每张图片下方有讨论区
- 支持 @mention 提及团队成员
- @mention 会触发站内通知

### 5.8 操作日志

- 记录所有项目操作：标注创建、图片上传、状态变更等
- 支持按操作类型筛选
- 支持撤销操作

### 5.9 图片对比

- 支持原图与修图后对比
- 对比模式：左右对比 / 滑块对比 / 叠加对比

### 5.10 图片下载

- 原尺寸下载
- Web 优化下载（2000px JPG，默认质量 85%）
- 自定义尺寸下载（自由设置宽高和质量）

## 6. 我的待办 — 修图任务

**路径**: `/my-tasks`

### 6.1 功能概述

修图师的任务管理中心，集中展示所有待处理的修图任务。

### 6.2 任务列表

- 按优先级排序：争议 > 临期 > 正常
- 显示项目名称、产品单元、任务描述、缩略图
- 标记已解决/未解决

### 6.3 筛选功能（V1.18 新增）

- **状态筛选**：全部 / 争议 / 临期 / 正常
- **项目筛选**：下拉选择特定项目
- 每个筛选项显示对应任务数量
- 筛选后实时更新列表

### 6.4 批量操作

- 支持批量标记为已解决

## 7. 客户分享端 — 品牌方审片

**路径**: `/client/project/:shareToken`

### 7.1 功能概述

客户通过分享链接免登录访问，进行审片确认和意见反馈。

### 7.2 品牌栏

- 显示工作室 Logo 和品牌名称
- 主题色跟随工作室品牌设置

### 7.3 审片进度

- 进度条显示已确认/已查看比例
- 每张图片可标记"已查看"
- 支持一键标记全部已查看

### 7.4 图片浏览

- 左右箭头切换图片
- 图片计数器显示当前位置
- 支持捏合缩放（移动端）
- 支持双击放大

### 7.5 意见反馈

- 每张图片对应意见卡片列表
- 确认（满意）/ 修改（不满意）双状态
- 修改时可填写具体修改要求
- 支持回复修图师的追问

### 7.6 AI 风格预览（V1.18 新增）

- 上传参考图
- 选择 4 种 AI 风格方向：暖色 / 冷色 / 自然光 / INS风
- 风格强度滑块（0-100%），实时预览效果
- 拖动滑块调整风格强度，图像即时变化

### 7.7 对比模式

- 左右对比：并排显示参考图和原图
- 滑块对比：拖动分割线对比
- 叠加对比：透明度叠加

### 7.8 确稿流程

- 审片完成后，点击"确认确稿"
- 二次确认弹窗防止误操作
- 确稿后修图师开始修图工作
- 修图完成后客户可查看修图对比

### 7.9 修改请求

- 确稿后如有不满意，可发起修改请求
- 填写修改说明
- 修图师收到通知后重新处理

## 8. 图片标注工具

### 8.1 工具类型

| 工具 | 快捷键 | 用途 |
|------|--------|------|
| 指针 | 默认 | 选择和移动标注 |
| 画笔 | `P` | 自由绘制线条标注问题区域 |
| 箭头 | `A` | 指向特定位置 |
| 矩形 | `R` | 框选问题区域 |
| 椭圆 | — | 圈选问题区域 |
| 文字 | `T` | 添加文字标注 |
| 橡皮擦 | — | 擦除标注 |

### 8.2 标注颜色

- 红色 `#FF0000`
- 黄色 `#FFCC00`
- 蓝色 `#0066FF`
- 白色 `#FFFFFF`

### 8.3 画笔宽度

- 细线 (3px)
- 中粗 (5px)
- 粗线 (10px)

### 8.4 文字大小

- 小号 (14px)
- 中号 (18px)
- 大号 (24px)

### 8.5 坐标系统

- 标注使用相对坐标系统（左上角 0,0，右下角 1,1）
- 标注在不同分辨率下保持位置一致

### 8.6 草稿恢复

- 未完成的标注自动保存为草稿
- 切换图片后返回可恢复草稿

## 9. AI 智能功能

### 9.1 AI 风格处理

- 上传图片后选择 AI 风格
- 6 种预设风格：北欧极简、波西米亚、冷调金属、自然光暖调、高级灰、原片增强
- 生成 4 张风格样本供选择
- 支持标记喜欢的样本
- 支持重试生成

### 9.2 AI 指令解析

- 客户自然语言修改意见自动解析为修图参数
- 如"脸部提亮一点" → 亮度 +10
- 修图师可确认或调整 AI 建议的参数

### 9.3 客户端 AI 风格预览（V1.18 新增）

- 客户上传参考图后选择风格方向
- 4 种方向：暖色 / 冷色 / 自然光 / INS风
- 强度滑块 0-100% 实时预览
- 通过 CSS filter 模拟效果

## 10. 巡检功能

### 10.1 色差巡检

**路径**: `/project/:id/color-check`

- AI 自动检测项目中所有图片的色彩偏差
- 检测维度：色温、亮度、对比度
- 生成异常图片列表
- 每张异常图片提供修图建议
- 支持一键创建修图任务

### 10.2 光影巡检

**路径**: `/project/:id/consistency-check`

- AI 检测整组图片的光影一致性
- 检测维度：光源方向、高光位置、阴影角度、色温、曝光、反射
- 自动分组场景
- 标记不一致的图片组
- 输出一致性评分（0-100 分）

## 11. 交付包

### 11.1 功能概述

项目完成后，可生成专业的交付包，包含所有修图成品和配套文件。

### 11.2 交付包内容

- **Excel 清单**：图片名称、所属单元、修图师、修改次数、状态
- **原图文件**：所有最终成品图片
- **PDF 报告**：项目总结报告
- **ZIP 打包**：以上内容的压缩包

### 11.3 下载方式

1. 在项目详情页点击"交付包"按钮
2. 查看交付清单确认内容
3. 点击下载生成 ZIP 文件

### 11.4 异步打包（V1.19 计划）

- 大文件（>50MB）自动切换为后台异步打包
- 打包完成后通知用户下载

## 12. 作品集

### 12.1 功能概述

将项目中的精选图片制作成 H5 作品集，支持分享给客户展示。

### 12.2 创建作品集

**路径**: `/project/:id/portfolio`

1. 选择项目中的图片
2. 设置封面图（支持自定义背景）
3. 填写客户名称和联系方式
4. 为每张图片添加描述
5. 支持拖拽排序

### 12.3 模板选择（V1.18 新增）

| 模板 | 布局 | 适用场景 |
|------|------|---------|
| 杂志风 | 瀑布流 (masonry) | 时尚/创意摄影 |
| 电商风 | 网格 (grid) | 电商产品图展示 |
| 极简风 | 单列 (single) | 简约风格展示 |
| 婚礼风 | 网格 (grid) | 婚礼/温馨场景 |

### 12.4 作品集查看

**路径**: `/portfolio/:id`

- 手机框架预览效果
- 记录查看次数和平均停留时长
- 支持分享链接

### 12.5 模板特性

- 每种模板有独立的配色方案（背景色、强调色、字体）
- 切换模板时保留已添加的图片和封面设置
- 支持手机端自适应

## 16. 键盘快捷键

### 16.1 图片查看器快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` / `Cmd+Enter` | 提交当前卡片（快速流转模式） |
| `A` | 切换到箭头工具 |
| `P` | 切换到画笔工具 |
| `R` | 切换到矩形工具 |
| `T` | 切换到文字工具 |
| `Esc` | 关闭当前弹窗/模态框 |
| `+` / `-` | 缩放图片 |
| `0` | 重置缩放 |
| `F` | 适应窗口 |
| `Ctrl+Z` | 撤销标注 |
| `Ctrl+Shift+Z` | 重做标注 |

### 16.2 项目详情页快捷键

| 快捷键 | 功能 |
|--------|------|
| `←` 左箭头 | 上一张图片 |
| `→` 右箭头 | 下一张图片 |

### 16.3 注意事项

- 快捷键仅在图片查看器聚焦且不在文本输入框时生效
- 工具切换键不区分大小写（`a`/`A` 均可）
