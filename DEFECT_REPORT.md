### DEF-001: 作品集跨 workspace 访问 — 已修复

| 字段 | 值 |
|------|-----|
| 缺陷ID | DEF-001 |
| 关联用例 | 17.3b |
| 严重等级 | P0 (致命) |

| 回归状态 | ✅ 通过 |
