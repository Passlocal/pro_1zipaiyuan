## 第一章 系统概述

### 1.1 平台定位

易拍选 (EpicShot) 是一款面向商业摄影工作室的**在线审片协作平台**。它将传统的"微信群发图 → 客户下载 → 截图标注 → 修图师改图"流程，升级为全在线的专业协作体验。

### 1.2 核心业务流程

```
摄影师拍摄 → 上传图片到项目 → 创建产品单元分类
    ↓
客户通过分享链接审片 → 标注问题区域 → 填写修改意见 → 确认确稿
    ↓
修图师收到任务 → 逐张修图 → 提交修图结果 → 标记已解决
    ↓
客户确认修图 → 工作室生成交付包 → 项目完成
```

### 1.3 三方角色视图

| 角色 | 主要入口 | 典型场景 |
|------|---------|---------|
| 老板 | 战情室 `/` | 看全局健康度、分配任务、查看财务 |
| 修图师 | 我的待办 `/my-tasks` | 处理分配给自己的修图任务 |
| 客户 | 分享链接 `/client/project/:token` | 免登录审片、标注修改意见 |

### 1.4 浏览器要求

- Chrome 90+ / Edge 90+ / Safari 15+
- 支持触屏设备（手机端标注功能）
- 推荐分辨率：1920×1080 或以上

## 第二章 登录与账号

### 2.1 登录步骤

1. 打开平台首页，系统自动跳转到登录页
2. 输入邮箱和密码
3. 点击"登录"按钮
4. 登录成功后，根据角色自动跳转：
   - **老板** → 战情室首页
   - **修图师** → 我的待办页面

### 2.2 注册账号

1. 在登录页点击"注册"链接
2. 填写邮箱、姓名、密码
3. 选择工作空间
4. 点击"注册"提交

### 2.3 角色说明

- **Owner（老板）**：拥有全部权限，可访问战情室、工作空间设置、所有项目
- **Editor（修图师）**：可访问项目、待办、作品集，不可访问战情室和工作空间

### 2.4 微信登录（可选）

平台支持微信扫码登录。需在"工作空间设置"中配置微信公众号后启用。

## 第四章 战情室

**仅老板可见** | 路径：`/`

### 4.1 页面布局

战情室从上到下分为四个区域：

```
┌─────────────────────────────────────────────┐
│  统计卡片行：总项目 | 活跃 | 红色预警 | 未解决  │
├─────────────────────────────────────────────┤
│  项目健康监控列表（表格）                       │
│  列：项目名 | 客户 | 状态 | 健康度 | 未解决数   │
├─────────────────────────────────────────────┤
│  成员负载面板                                 │
│  列：成员名 | 头像 | 任务数 | 展开详情         │
├─────────────────────────────────────────────┤
│  财务分析面板                                 │
│  合同金额统计 | 月收入趋势 | 阶段付款汇总      │
└─────────────────────────────────────────────┘
```

### 4.2 项目健康状态

每个项目显示一个健康指示灯：

- **绿色**：正常，争议卡片 < 阈值，无临期风险
- **黄色**：预警，争议卡片 ≥ 阈值，或临近截止日期（7天内）
- **红色**：严重，大量争议卡片，或已超过截止日期

健康原因会以文字提示方式显示在项目行中。

### 4.3 成员负载

- 每个成员显示当前任务总数
- 点击成员行可展开，查看该成员在各项目中的具体任务
- 临期任务会标注橙色"临期"标签

### 4.4 财务分析

- **合同金额汇总**：所有项目的合同金额总和
- **月度趋势**：按月份统计收入
- **阶段付款**：按定金/中期款/尾款等阶段汇总（V1.18 新增）
- 支持导出 Excel 格式

### 4.5 数据导出

点击"导出"按钮，系统将当前仪表盘数据导出为 Excel 文件，包含：
- 项目健康统计表
- 成员负载统计表
- 财务数据汇总表

### 5.2 创建项目 — 详细步骤

**第一步：打开创建弹窗**

点击页面右上角"+ 创建项目"按钮，弹出创建表单。

**第二步：填写基本信息**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 项目名称 | 文本输入 | 是 | 不超过200字符 |
| 客户名称 | 文本输入 | 否 | 品牌方客户名称 |
| 截止日期 | 日期选择 | 否 | 项目预期完成日期 |
| 合同金额 | 数字输入 | 否 | 单位：元，支持2位小数 |
| 备注 | 文本输入 | 否 | 项目特殊说明 |

**第三步：设置付款阶段（V1.18 新增）**

1. 填写合同金额后，点击"+ 添加付款阶段"按钮
2. 输入阶段名称（如"定金"、"中期款"、"尾款"）
3. 输入阶段金额
4. 系统自动计算该阶段占总金额的百分比
5. 可添加最多 5 个阶段
6. 如果阶段金额合计与总金额不一致，系统显示红色警告：
   ```
   阶段合计 ¥70,000，与总金额 ¥100,000 不一致
   ```
7. 点击阶段右侧的"×"按钮可删除该阶段

**第四步：确认创建**

点击"确认创建"按钮提交。系统会验证：
- 项目名称不能为空
- 项目名称不超过200字符
- 创建成功后自动跳转到项目详情页

### 5.3 项目列表操作

**搜索项目**

在搜索框中输入关键词，系统实时按项目名称和客户名称匹配。

**状态筛选**

点击状态标签可筛选特定状态的项目。状态包括：
- 草稿（draft）— 刚创建，尚未分享
- 待确认（review）— 已分享给客户，等待客户审片
- 修改中（in_progress）— 客户已确稿，修图师正在修图
- 待确稿（final_review）— 修图完成，等待客户最终确认
- 已完成（completed）— 全部确认通过
- 已归档（archived）— 已归档的项目

**项目卡片操作**

点击项目卡片进入项目详情页。每个卡片显示：
- 项目封面图（可点击更换）
- 项目名称
- 客户名称
- 状态标签
- 待处理卡片数量
- 最近活动时间

### 5.4 项目模板

**保存为模板**

在项目详情页的"更多"菜单中，可将当前项目的结构（产品单元分类、设置）保存为模板。

**从模板创建**

在创建项目时，可选择已有模板快速创建，自动复用模板中的产品单元结构。

### 6.3 产品单元管理

**什么是产品单元**

产品单元是项目的分类维度，通常对应不同的 SKU、场景或品类。例如：
- 服装项目：红色款 / 蓝色款 / 黑色款
- 美食项目：前菜 / 主菜 / 甜品
- 场景项目：室内 / 室外 / 特写

**创建产品单元**

1. 在右侧面板顶部点击"+ 添加产品单元"
2. 输入单元名称
3. 按 Enter 确认，或点击外部取消

**管理产品单元**

- 点击单元名称可展开/折叠该单元下的图片
- 支持拖拽排列单元顺序
- 点击单元可查看该单元下的所有图片

**上传图片**

1. 选中目标产品单元
2. 点击"上传图片"或拖拽图片到上传区域
3. 支持批量选择多张图片
4. 上传完成后自动生成多级缩略图（320px / 800px / 原图）

### 6.4 图片查看器

**图片切换**

- 点击右侧缩略图切换到对应图片
- 使用键盘 `←` `→` 方向键切换
- 点击图片查看器下方的导航按钮

**图片缩放**

- 使用 `+` / `-` 键缩放
- 按 `0` 重置缩放
- 按 `F` 适应窗口
- 鼠标滚轮缩放
- 移动端支持双指捏合缩放

**图片对比**

点击图片查看器中的对比按钮，可选择三种模式：
- **左右对比**：并排显示原图和处理后的图片
- **滑块对比**：拖动中间分割线，左侧原图、右侧处理图
- **叠加对比**：处理图叠加在原图上，可调整透明度

### 6.5 意见卡片管理

**创建意见卡片**

1. 在图片上使用标注工具标注问题区域
2. 在右侧卡片面板中填写修改意见
3. 点击"提交"创建卡片

**卡片状态**

- **未解决（unresolved）**：修图师尚未处理
- **已解决（resolved）**：修图师已完成修图

**卡片操作**

| 操作 | 按钮 | 说明 |
|------|------|------|
| 解决 | 标记为已解决 | 修图师确认修图完成 |
| 争议 | 标记争议 | 对修图结果不满意，重新打开 |
| 指派 | 指派给... | 将卡片分配给特定修图师 |
| 追问 | 回复区域 | 修图师向客户追问修改细节 |

**争议机制**

- 同一卡片被标记为"争议"超过设定阈值时，自动触发预警
- 争议卡片在战情室中计入健康度计算
- 争议卡片在待办列表中优先显示

### 6.6 快速流转模式

快速流转模式用于修图师逐张高效处理卡片。

**操作流程**

1. 确保"提交确认"开关处于期望状态（开启后提交前二次确认）
2. 查看当前图片的卡片，完成修图
3. 点击"提交"按钮（或按 `Ctrl+Enter`）
4. 系统自动切换到下一张有卡片的图片
5. 图片预加载机制确保切换无白屏闪烁（V1.18 新增）

**提交确认开关**

点击顶部工具栏的"提交确认"按钮可切换：
- **开启**（蓝色高亮）：提交前弹出确认弹窗，防止误操作
- **关闭**：直接提交，适合熟练操作

### 6.7 讨论区

每张图片下方有独立的讨论区。

**使用 @mention**

1. 在讨论区输入框中输入 `@`
2. 系统弹出成员列表供选择
3. 选择目标成员后继续输入内容
4. 发送后，被提及的成员会收到站内通知

**讨论区功能**

- 支持查看历史讨论记录
- 每条讨论显示发送者头像、姓名和时间
- 被提及的成员名以蓝色高亮显示

### 6.8 操作日志

**打开操作日志**

点击顶部工具栏"操作日志"按钮，弹出操作日志弹窗。

**筛选日志**

弹窗顶部提供筛选标签：
- 全部
- 标注
- 上传
- 状态变更
- 其他

**日志内容**

每条日志记录包含：
- 操作类型（标注创建 / 图片上传 / 状态变更等）
- 操作人
- 操作描述
- 操作时间

**撤销操作**

支持撤销部分操作（如误操作的状态变更）。

### 6.9 图片下载

**下载选项**

点击顶部"下载"按钮旁的下拉箭头，展开下载菜单：

| 选项 | 说明 |
|------|------|
| 原尺寸下载 | 下载原始大小的图片文件 |
| Web 优化 (2000px JPG) | 下载缩放到 2000px 宽的 JPG 格式，质量 85% |
| 自定义尺寸 | 自定义宽度（px）和压缩质量（1-100%） |

**自定义下载**

1. 选择"自定义尺寸"
2. 输入目标宽度（默认 2000px）
3. 调整压缩质量滑块（默认 85%）
4. 点击"下载"

### 6.10 抽查功能

**进入抽查**

点击"更多" → "抽查"，进入抽查模式。

**抽查流程**

1. 系统随机展示项目中的图片
2. 修图师/老板逐张检查修图质量
3. 可标记"通过"或"不通过"
4. 抽查结果记录在操作日志中

### 7.2 颜色选择

工具栏提供 4 种预设颜色按钮：

| 颜色 | 色值 | 推荐用途 |
|------|------|---------|
| 红色 | `#FF0000` | 需要修改的问题 |
| 黄色 | `#FFCC00` | 需要注意/待讨论 |
| 蓝色 | `#0066FF` | 信息提示 |
| 白色 | `#FFFFFF` | 深色图片上的标注 |

点击颜色按钮切换当前颜色，后续标注将使用新颜色。

### 7.3 画笔宽度

画笔工具提供 3 种宽度选项：

- **细线 (3px)**：精细标注
- **中粗 (5px)**：常规标注（默认）
- **粗线 (10px)**：醒目标注

### 7.4 文字大小

文字工具提供 3 种字号：

- **小号 (14px)**：细节标注
- **中号 (18px)**：常规标注（默认）
- **大号 (24px)**：标题标注

### 7.5 标注操作

**创建标注**

1. 选择工具（如画笔）
2. 在图片上绘制
3. 释放鼠标/手指完成标注

**移动标注**

1. 切换到"指针"工具
2. 点击选中标注
3. 拖拽移动位置

**删除标注**

- 选中标注后按 Delete 键
- 或使用橡皮擦工具点击标注

**撤销/重做**

- `Ctrl+Z`：撤销上一步
- `Ctrl+Shift+Z`：重做

### 7.6 标注草稿恢复

- 未提交的标注自动保存为草稿（存储在浏览器 localStorage）
- 切换图片后再返回，系统提示"检测到未完成的标注，是否恢复？"
- 点击"恢复"重新加载草稿内容
- 点击"忽略"清除草稿

### 7.7 相对坐标系统

标注使用相对坐标系统（0-1）：
- 左上角：(0, 0)
- 右下角：(1, 1)
- 坐标与图片实际像素无关
- 同一标注在不同分辨率设备上位置一致

### 16.2 查看通知

## 第十八章 常见问题与故障排除

### 18.1 图片相关

**Q：上传图片失败怎么办？**

- 检查图片格式是否支持（JPG、PNG、WEBP、TIFF）
- 检查单张图片大小是否超过 50MB
- 检查网络连接是否稳定
- 尝试分批次上传

**Q：图片加载缓慢或显示白屏？**

- 检查网络带宽
- 尝试刷新页面
- 系统已启用图片预加载机制（V1.18），通常切换图片无白屏

**Q：图片显示颜色与预期不符？**

- 确认浏览器是否支持色彩管理
- 使用色差巡检功能检查图片色彩偏差

### 18.2 标注相关

**Q：标注位置和其他设备不一致？**

- 系统使用相对坐标系统，标注应在不同设备上位置一致
- 如不一致，请检查浏览器缩放比例是否 100%

**Q：标注意外丢失？**

- 检查是否误触了橡皮擦或 Delete 键
- 使用 Ctrl+Z 尝试撤销
- 检查草稿恢复功能是否保存了未提交的标注

### 18.3 分享相关

**Q：客户无法打开分享链接？**

- 检查分享是否已过期（可在项目设置中查看有效期）
- 检查链接是否完整复制
- 重新生成分享链接

**Q：客户看不到最新上传的图片？**

- 分享链接是实时同步的，刷新页面即可看到最新图片
- 如仍看不到，确认图片是否已上传到正确的产品单元

### 18.4 交付包相关

**Q：交付包下载失败？**

- 检查项目是否包含图片
- 确认网络连接正常
- 大项目文件较大，请耐心等待打包完成

**Q：交付包中 Excel 格式异常？**

- 确保使用 Excel 2016 或更高版本打开
- 或使用 WPS、Google Sheets 打开

### 18.5 账号相关

**Q：忘记密码？**

- 联系工作室管理员重置密码

**Q：无法登录？**

- 检查邮箱和密码是否正确
- 确认账号未被禁用
- 清除浏览器缓存后重试
